# Food Around The World

a food study project that scrapes food names and its search index on Google Trend's API and generates analysis \& visualization based on a trend's significant patterns. The project is the semester-long assignment for the Computational Analysis of Big Data Course. 

## Built With

* [pytrends] (https://github.com/GeneralMills/pytrends) - The API used to scrape trend index on Google Trends
* [Jupyter Notebook] - The platform used to process and analyze data
* [Plot.ly] - The library used for visualization
